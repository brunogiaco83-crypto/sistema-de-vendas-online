 Timon Turismo

Aplicativo de turismo desenvolvido para promover os atrativos turísticos, culturais e gastronômicos do município de Timon – MA.

O projeto contempla prototipação de interface, modelagem de banco de dados relacional e implementação da base de dados utilizando MySQL.

##  Sobre o Projeto

O Turismo Timon tem como objetivo facilitar o acesso às informações turísticas do município, oferecendo:

 Pontos turísticos

 Eventos culturais

 Gastronomia local

 Hospedagens

 Localização via mapa

 Galeria de imagens

## O sistema foi estruturado para futura expansão e integração com backend e aplicativo mobile.

 Tecnologias Utilizadas

 Figma – Prototipação da interface

 dbdiagram.io – Modelagem do banco de dados

 MySQL – Sistema de gerenciamento de banco de dados relacional
